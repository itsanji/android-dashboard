## Create a React Native app that does the following:

### App has these main screens:
#### 1. Dashboard screen
- It is a Dashboard app. When the user opens the app, it shows widgets that the user has configured. 
- The widgets can be any of the following:
  - Calendar: events retrieved from the phone's calendar
  - Weather: retrieved from the phone's weather service
  - Media controller: controls currently playing media
  - Custom Text
  - Clock with styles
- Background is customizable. It can be either an image or a video.
- Button to navigate to the settings screen.

#### 2. Settings screen
- Has a Widgets list tab where users can click to add widgets to the dashboard screen.
- Has an added widgets list where users can click on a widget to manage: widget position, widget content, or styles.
- Button to return to the Dashboard screen. 


## Tech Stack:
- React Native
- Expo.
- Shizuku api (if needed)
